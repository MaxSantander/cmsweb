API Endpoints
============

Overview
--------
Documentation of available API endpoints in the RIS CMS.

Core Endpoints
------------

Page Management
~~~~~~~~~~~~~
.. code-block:: text

   GET /api/pages/
   POST /api/pages/
   GET /api/pages/<id>/
   PUT /api/pages/<id>/
   DELETE /api/pages/<id>/

Block Management
~~~~~~~~~~~~~
.. code-block:: text

   GET /api/blocks/
   POST /api/blocks/
   GET /api/blocks/<id>/
   PUT /api/blocks/<id>/
   DELETE /api/blocks/<id>/
